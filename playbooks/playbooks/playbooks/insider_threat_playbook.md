# 🚨 SOC Playbook: Insider Threat Investigation

## 📌 Overview
This playbook addresses suspicious or malicious activity originating from internal users.

---

## 🎯 Objective
- Detect insider misuse
- Investigate abnormal behavior
- Protect sensitive data

---

## ⚠️ Indicators
- Unusual login times
- Large data transfers
- Access to unauthorized systems
- Privilege escalation

---

## 🔍 Phase 1: Triage
- Review alert details
- Identify user and activity
- Determine risk level

---

## 🧪 Phase 2: Investigation
- Analyze login patterns
- Review file access logs
- Identify data movement
- Check privilege changes

---

## 🛑 Phase 3: Containment
- Restrict user access
- Monitor account activity
- Escalate if necessary

---

## 🧹 Phase 4: Eradication
- Remove unauthorized access
- Reset credentials
- Disable accounts if required

---

## 🔄 Phase 5: Recovery
- Restore normal access controls
- Monitor user behavior

---

## 📘 Phase 6: Lessons Learned
- Improve access controls
- Enhance logging and monitoring
- Update user policies

---

## ⚠️ Escalation Criteria
- Data exfiltration suspected
- Privileged account misuse
- Legal/compliance implications

---

## 🧬 MITRE ATT&CK
- T1078 – Valid Accounts
- T1020 – Automated Exfiltration
